# PT Pandawa Mitra Indonesia — Website

## Struktur File

```
pandawa-website/
├── index.html          # Halaman utama
├── css/
│   └── style.css       # Semua styling
├── js/
│   └── main.js         # Interaktivitas & animasi
└── README.md
```

## Cara Menggunakan

1. **Buka Langsung**: Klik dua kali `index.html` untuk membuka di browser
2. **Via Live Server**: Gunakan VS Code Live Server extension untuk preview optimal

## Fitur Website

- ✅ Responsive (mobile, tablet, desktop)
- ✅ Navbar sticky dengan efek scroll
- ✅ Hero section dengan animasi
- ✅ Section Tentang + Visi Misi
- ✅ Section Layanan (6 kartu)
- ✅ Flow Proses Penempatan PMI (6 langkah)
- ✅ Struktur Organisasi + RACI Matrix
- ✅ Section Keunggulan
- ✅ Form Kontak interaktif
- ✅ Footer lengkap
- ✅ Smooth scroll & reveal animations
- ✅ Mobile hamburger menu

## Kustomisasi

### Ganti Warna
Edit CSS variables di `css/style.css` bagian `:root`:
```css
--navy: #0c1f4a;      /* Warna utama navy */
--gold: #c8a84b;      /* Warna aksen emas */
```

### Update Konten
Edit langsung di `index.html` — semua teks mudah ditemukan dengan komentar section.

### Font
Menggunakan Google Fonts:
- **Cormorant Garamond** — heading/display (elegant serif)
- **DM Sans** — body text (clean sans-serif)

## Teknologi

- HTML5 + CSS3 + Vanilla JavaScript
- Google Fonts
- IntersectionObserver API (animasi scroll)
- No framework/library — lightweight & fast
